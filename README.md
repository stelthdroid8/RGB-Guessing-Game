# RGB-Guessing-Game
Creates a game that chooses a random rgb color selection for the user to view and correlate those numbers into a visual color they see.

This game starts off with only 2 modes, easy and hard. Easy contains only 3 possible choices and hard contains 6.

The goal of this game is to figure out what the color that is shown in rgb(xxx,xxx,xxx) form looks like visually. With n-1 wrong answers also randomly generated, where n is the number of total squares generated.

Each color displayed is chosen at random, and from those random colors it chooses 1 of them, also at random, to be the correct color. After it chooses the correct color it shows it to the user in rgb format and the player has to choose from the given colors which one is correct.
